# Cara Kerja Program (Versi Orang Awam)

Dokumen ini menjelaskan cara kerja aplikasi absensi dengan bahasa sederhana.

## Gambaran Singkat

Bayangkan aplikasi ini seperti buku hadir digital yang lebih rapi dan otomatis.

- Siswa login seperti membuka buku hadir.
- Jika hadir, siswa menekan tombol absen pagi atau sore.
- Jika tidak hadir, siswa mengisi alasan izin.
- Admin melihat semua data dan bisa menyetujui atau menolak izin.
- Semua catatan tersimpan otomatis dan rapi.

## 1. Login

1. Siswa memasukkan nama dan password.
2. Sistem memeriksa apakah data benar.
3. Jika benar, siswa bisa masuk ke aplikasi.

## 2. Absen Hadir

1. Siswa memilih absen pagi atau sore.
2. Sistem mencatat jam masuk secara otomatis.
3. Jika sudah absen untuk sesi itu, siswa tidak bisa absen dua kali.

## 3. Ajukan Izin

1. Siswa mengisi alasan izin.
2. Sistem menyimpan alasan tersebut.
3. Status izin akan menunggu persetujuan admin.

## 4. Persetujuan Admin

1. Admin membuka daftar izin.
2. Admin bisa menekan tombol Setujui atau Tolak.
3. Hasilnya langsung terlihat di aplikasi.

## 5. Laporan Absensi

1. Admin memilih tanggal, kelompok, dan nama siswa.
2. Sistem menampilkan data yang sesuai.
3. Data bisa diunduh sebagai file laporan.

## Kenapa Waktu Diatur oleh Sistem

- Waktu absen ditentukan oleh server agar tidak bisa diubah oleh jam handphone.
- Tampilan waktu di layar dibuat rapi agar mudah dibaca.

## Ringkasnya

Aplikasi ini membantu mencatat kehadiran dan izin dengan lebih cepat, aman, dan rapi. Siswa cukup klik tombol, admin cukup memantau dan menyetujui.
